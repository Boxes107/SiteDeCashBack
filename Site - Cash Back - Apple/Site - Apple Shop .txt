# Sistemas : 

- Login :
  Usuario (Sistema de Cadastro)
  Senha
  Conexão com redes sociais 

- Seleção e Compra de Dominio
  - Administração de Dominio
  - Campos de Redirecionamento de Dominio e Sub Dominio  

- Cadastro de Produtos - Banco de Dados Firebase 
  - Campo de Inserção de Valor
  - Cálculo do CashBack
  - Registro do CashBack no Banco de Dados do Cliente
  - Banco de Dados Separado para Cada Cliente 
  - Plataforma SQL Unica para Todos os Clientes
  - Registro da Plataforma em Nuvem - Assinatura na Nuvem

- Modelo de Site - Projeto CSS - Javascript - SQL 
  - Campo do Site Inicial - Página Inicial
  - Campo da Página Inicial Pós Login para Usúario Comum
    - Informações de CashBack Dispóniveis por Produto
    - CashBack Dispónivel por Acumulo
    - Possibilidade de Acumulo por Indicação e Valor de Compra da Indicação ?
  - Pagina do Administrador
    - Campos da Página do Administrador
    - Comandos de Admin Separados para Administrador
    - Comandos de Admin Separados para Gerenciamento do Programador
    
- Sistema Futuro Aplicativo Android e IOS - Flutter - Java
  - Preparação de Código para Base Mutua
  - Utilização de Plataformas de Código Misto 
  - Sistema de Importação Simples Para Aplicativo
  - Sistema de Anuncio de Promoções
    - Criação de Plataforma para Gerenciamento do Designer 
      - Plataforma Simples com Campos Base Formados 
      


- Pagamento Online Retirada na Loja ou por Delivery 
  - Sistema de Ciclagem de Postagens Baseado no Counter de Clientes
    - Sistema de Monitoramento de Novos e Antigos Clientes

- Sistema de Ficha
